install xampp

install composer

set up system variables (add System Variable "PHP" name and "C:\xampp\php\php.exe" and "Composer" name with value "C:\ProgramData\ComposerSetup\bin")

check composer version in cmd with "composer --version" should show composer version and PHP version

download laravel by entering into terminal prompt "composer global require laravel/installer"

uncomment extension=fileinfo, extension=zip, extension=mysqli
comment extension=openssl in xampp/bin/ini.php

create new laravel project by entering laravel new firstwebsite

open project folder

change APP_URL in .env file to "http://localhost/dashboard/firstwebsite/public"

visit route and change home route

running "npm install vite --save-dev" in terminal to start vite

# While coding (see changes live)
npm run dev

# When ready to test like real users will see it
npm run build    # Make optimized version
npm run preview  # Test that optimized version

migrate controller resources:
php artisan make:model Post -mcr

other codes:
# Model + Migration only
php artisan make:model Post -m

# Model + Migration + Controller (non-resource)
php artisan make:model Post -mc

# Model + Migration + Factory + Seeder + Controller
php artisan make:model Post -mfscr

# All common components (common shortcut)
php artisan make:model Post -a
# Equivalent to: -mfsrc (migration, factory, seeder, resource controller)

run migrations:
php artisan migrate

undo:
php artisan migrate:rollback

undo specific migrated file:
php artisan migrate:rollback --path=database/migrations/2026_02_03_075146_create_posts_table.php

migrate specific file
php artisan migrate --path=database/migrations/2026_02_03_075146_create_posts_table.php

migrate all files, rerun all of them:
php artisan migrate:refresh
